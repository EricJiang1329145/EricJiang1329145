这个主题已经安装并配置了 [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) 集成，并在你的 `astro.config.mjs` 配置文件中。如果你不想使用 MDX，可以通过从配置文件中删除集成来禁用支持。

## 为什么要使用 MDX？

MDX 是一种特殊的 Markdown 格式，支持嵌入 JavaScript 和 JSX 语法。这解锁了[在 Markdown 内容中混合 JavaScript 和 UI 组件](https://docs.astro.build/en/guides/markdown-content/#mdx-features) 的能力，用于实现交互式图表或警报等功能。

如果你有现有的 MDX 创作内容，这个集成将帮助你轻松迁移到 Astro。
